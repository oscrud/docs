Request Context

MethodDescription
Method() stringReturn request method, default to be smaller case get, post.
Get(key string) interface{}Get value by key from param, query, body, header, order respectively.
Context() context.ContextGet request context
Transport() TransportIDGet transport name
Path() stringReturn request path
RequestID() stringReturn request id
State() map[string]interface{}Return request state
Headers() map[string]stringReturn request headers
Query() map[string]interface{}Return request queries
Params() map[string]stringReturn request params
Body() map[string]interface{}Return request body
Bind(src interface{}) errorBind data from map specify by reflect.Tag. More information please look at Specific Binding.
BindAll(src interface{}) errorBind data from param, query, body, header, state based on json and qm tag. More information please look at All Binding.
SetState(key string, value interface{})Set data to request level state
